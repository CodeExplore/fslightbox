# Fullscreen Lightbox Basic

## Version - 1.4.3

## Description
JavaScript media box library for presenting images, videos.

Open source project.
Technologies:
- JavaScript
- SCSS
### No jQuery and other dependencies.

## Instalation
Clone repository cd to directory and run:
````
 npm install
 ````
 
To run gulp server with tasks:
````
 gulp
 ````
